# CyberToolkit
A simple cybersecurity toolkit built with C++ for learning purposes.
# Cyber Security Toolkit

## Features
- Password Strength Checker
- Caesar Cipher Encryption
- Caesar Cipher Decryption
- Password Generator
- Login System
- Port Scanner Simulation

## Technologies Used
- C++
- Standard Library

## Purpose
This project was created for learning purposes and to improve understanding of basic cybersecurity and C++ programming concepts.

## How to Run

Compile:
```bash
g++ CyberToolkit.cpp -o toolkit
```

Run:
```bash
./toolkit
```
